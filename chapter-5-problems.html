    Example 1.0.1. Integration Techniques.

    We have learned about four primary techniques of integration this semester:
    • Integration by Substitution
      🔗
    • Integration by Parts
      🔗
    • Partial Fraction Decomposition
      🔗
    • Trigonometric Substitution
      🔗
    Consider the following integral:
    \begin{equation*} \int \frac{x}{\sqrt{x^2+9}} \, dx \end{equation*}
    In the space below, justify how you would use two of the techniques of integration listed above to solve the given integral. For each technique, you should say in a sentence why the technique is applicable to the given integral, then indicate the first step or substitution that you would use accordingly. You do not need to actually solve the integral using either technique.
    🔗
    Solution.
    • The denominator is of the form \(\sqrt{x^2+a}\text{,}\) where \(a>0\text{,}\) so we can do the Trigonometric Substitution \(x=\sqrt{a}\tan(\theta)\text{.}\) In this instance, \(a=9\text{,}\) so we substitute \(x=3\tan(\theta)\text{.}\)
      🔗
      🔗
    • The derivative of the argument of the square root is present in the numerator up to a constant multiple, so we can perform an Integration by Substitution. That is, if we differentiate \(x^2+9\) with respect to \(x\text{,}\) we get \(2x\text{,}\) which is in the numerator if we divide it by the constant \(2\text{.}\) This allows the \(u\)-substitution \(u=x^2+9\) to completely rewrite the integral in terms of \(u\text{.}\)
      🔗
      🔗

    Example 1.0.2. Partial Fraction Decomposition.

    Without solving for any coefficients, apply partial fraction decomposition to the following rational expression:
    \begin{equation*} \frac{1}{(x^2+7)(x^2+4x+3)}. \end{equation*}
    Your answer should be a sum of fractions whose numerators include coefficients - coefficients you do not need to solve for.
    🔗
    Solution.
    We first factor the denominator. Note that \(x^2+7\) is irreducible (it does not factor), while
    \begin{equation*} x^2+4x+3=(x+3)(x+1). \end{equation*}
    So we have
    \begin{equation*} \frac{1}{(x^2+7)(x^2+4x+3)}=\frac{1}{(x^2+7)(x+3)(x+1)}. \end{equation*}
    Now, again since \(x^2+7\) is an irreducible quadratic, the numerator of its term in the partial fraction decomposition should have degree one less than the degree of it. That is, the numerator should be degree \(2-1=1\text{.}\) A general degree 1 polynomial is of the form \(Ax+B\text{,}\) for some \(A\) and \(B\text{.}\) The two linear factors \((x+3)\) and \((x+1)\) are similar. They are degree 1 polynomials, so the numerator of the term in their partial fraction should have degree \(1-1=0\text{.}\) Degree 0 polynomials are constants, say \(C\) and \(D\text{.}\) This gives the decomposition
    \begin{equation*} \frac{1}{(x^2+7)(x+3)(x+1)} = \frac{Ax+B}{x^2+7} + \frac{C}{x+3} + \frac{D}{x+1}. \end{equation*}

    Example 1.0.3. Improper Integrals.

    Determine the value of the following improper integral (or show that the integral diverges):
    \begin{equation*} \int_{10}^{\infty}\frac{1}{x^3}\,dx \end{equation*}
    Note: Your work will be graded for mathematical accuracy, so be clear and careful about using limit notation as relevant.
    🔗
    Solution.
    \begin{align*} \lim_{b\to\infty} \int_{10}^{b}\frac{1}{x^3}\,dx \amp = \lim_{b\to\infty}\int_{10}^{b}x^{-3}\,dx\\ \amp = \lim_{b\to\infty}\left[\frac{x^{-2}}{-2}\right]_{10}^{b}\\ \amp = -\frac12\lim_{b\to\infty}\left(\frac{1}{b^2}-\frac{1}{10^2}\right)\\ \amp = -\frac12\left(0-\frac{1}{100}\right)\\ \amp = \frac{1}{200}. \end{align*}
